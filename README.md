# Botswana Sovereign Risk & Macroeconomic Investment Analysis

Sovereign risk and macroeconomic investment analysis of Botswana using Excel-based data modeling.

## Overview
This project evaluates Botswana as a medium- to long-term investment opportunity using sovereign risk, macroeconomic, institutional, and commodity-exposure analysis.

## Key Findings
- Botswana has strong institutional quality, political stability, and relatively low public debt.
- The country shows long-term growth potential supported by capital accumulation and trade openness.
- Major risks include diamond-sector dependence, high unemployment, currency exposure, and recent sovereign credit rating downgrades.
- Final recommendation: long-term investment opportunity, with short-term caution due to diamond-market volatility.

## Methods Used
- Sovereign risk analysis
- Business cycle analysis
- Solow growth model
- Endogenous growth framework
- Credit-risk assessment
- Currency and commodity-risk analysis
- Excel-based data visualization

## Tools & Data Sources
- Microsoft Excel
- World Bank Open Data
- Bank of Botswana data
- S&P / Moody’s credit rating information
- OEC trade data
- Diamond price index data
